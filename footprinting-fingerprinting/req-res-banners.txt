BANNERS

-> São mensagens/informações num formato de cabeçalho
-> Eles são enviados por servidores de uma aplicação web como resposta à uma requisição
-> Fornecem informações sobre:
    -> versão do servidor web (apache/2.4.29)
    -> copyright
    -> infos de contato do administrador do servidor
    -> políticas de privacidade
    -> etc

-> BANNERS DE SERVIDORES WEB
    -> nc site.com 80 (+ qualquer texto depois)
    -> envia esse cabeçalho

-> BANNERS DE CONEXOES SSH
    -> service ssh start (abre um serviço ssh na porta 22 (que é a default para conexoes ssh))
    -> nc 127.0.0.1 22
    -> retorna o banner ssh

-> BANNERS DE CONEXOES COM BANCO DE DADOS
    -> service mysql start (inicia um servidor mysql na porta 3306)
    -> nc 127.0.0.1 3306
    -> retorna o banner
