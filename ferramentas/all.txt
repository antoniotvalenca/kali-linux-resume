NETCAT (linha de comando)
-> cria conexões entre sistemas e permite troca de dados (através da abertura de portas)

NGROK (linha de comando)
-> permite que pessoas de outras máquinas tenham acesso a uma aplicação que esteja rodando no seu localhost

NMAP (linha de comando)
-> faz port scanning, verifica quais serviços estao rodando em tal porta, etc
-> preferencia pela versao 7.92 pq a versao 7.93 ta com erro de openssl

CURL/WGET (linha de comando)
-> permite transferencia de dados atraves de diversos protocolos de rede (http, https, ftp, smtp...)

WHOIS (linha de comando) (site)
-> faz fingerprinting de um domínio/servidor (informações sociais, ip, etc)

FIERCE (linha de comando)
-> faz pesquisa DNS em um cominio especifico (registro de hosts, subdominios, etc)

WADW00F (linha de comando)
-> detecta firewalls

CRT.SH (site)
-> encontra subdominios de um domínio principal

DIRB (linha de comando)
-> faz um bruteforce de URLs

SQLMAP (linha de comando)
-> faz sqlinjection para descobrir infos do banco de dados (nome do banco/de tabelas/ de colunas/de dados)

EXIFTOOLS (linha de comando)
-> mexe com metadados de arquivos/imagens etc
-> usado para inserçao de codigo malicioso dentro de imagens (um arquivo legitimo que camufla o codigo) para webshells

HASH-IDENTIFIER (linha de comando)
-> usado para identificar qual o tipo mais possível da hash
-> voce cola a hash e ele te retorna o resultado

BURP SUITE (software)
